# Security Policy

Most components in this repository are currently in beta and primarily target testnet environments.

If you discover a security vulnerability, please report it privately to:

- security@digitalguards.nl

Please include clear reproduction steps, impact assessment, and affected version/commit details where possible.

Do not publicly disclose security issues before coordination with the maintainers.

Severe, validated findings may be eligible for a QRL bounty at the discretion of Digital Guards.
